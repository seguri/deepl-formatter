# deepl-formatter

An extension to remove random newline and add them in the right places (after `?`, `!`, and `.`).

## Requirements

Packages:

    npm i -g yo generator-code vsce

## Installation

Create the vsix:

    vsce package

In vscode:

    Command Palette (⇧⌘P) > "vsix" > Install from VSIX > Select the vsix in this folder
